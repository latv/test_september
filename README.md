Programēšana vēlos sasniegt daudz, lai es varētu izveidot dažadas sarezģītibas lapas, kas pielāgojas ekrāna izmēram. Iemācīties Wordpresu un varbūt uzlabot savas zināšanas React un iemācīties Redux.
Programēšanu izvēlējos, jo tur var daudz ko macīties , gandrīz katru dienu iemācos daudz ko jaunu.
